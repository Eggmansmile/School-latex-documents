1.1.2 Integratie
Integratie en differentiatie zijn omgekeerde operaties. Als f (t ) de afgeleide is van g (t ), i.e., f (t ) = g '(t ), dan noemen we g (t ) de primitieve functie van f (t ). Uit de differentiatie-eigenschappen voor de afgeleide van een constante functie en voor de afgeleide van een som van functies (zie hoger) kunnen we dan een-
voudig inzien dat, wanneer g (t ) een primitieve functie is van f (t ), ook g (t ) + c een primitieve functie is van f (t ) voor elke constante waarde c ? R, aangezien er geldt dat (g (t ) + c)' = g '(t ) + 0 = f (t ). Met andere woorden, de primitieve functie
van een gegeven functie is niet uniek en kan enkel bepaald worden op een con- stante term na. We benoemen de verzameling van alle primitieve functies van een gegeven functie als de onbepaalde integraal van die functie, en we schrijven Z f (t ) dt = (c)g (t ) + c, ?c ? Rª ? g '(t ) = f (t )

Merk op dat tabellen met afgeleiden en primitieve functies van veel voorkomende functies in heel wat tek- stboeken kunnen worden teruggevonden. 2.1 Beknopte theorie
Lineaire tijdsinvariante continue systemen kunnen beschreven worden d.m.v. gewone lineaire differentiaalvergelijkingen (LDVs) met constante coëfficieën- ten. Dit zijn vergelijkingen van de vorm

De oplossing y (t ) voor een LDV is doorgaans niet uniek, en bestaat in een uitdrukking die afhangt van enkele constanten die vrij gekozen kunnen worden. Om een unieke oplossing te vinden, wordt de LDV meestal uitgebreid met beginvoorwaarden die de waarde op tijdstip t = 0 uit-
drukken van de output y (t ) en van de afgeleiden dk y (t ) van de output tot en met orde N - 1. Een dergelijk probleem, bestaande uit een LDV met beginvoorwaar- den, wordt een beginwaardeprobleem genoemd. We noemen N de orde van het probleem.
    In deze beknopte theorie zullen we de klassieke methode toelichten voor het oplossen van LDVs met constante coëfficieënten, alsook voor de bijhorende beginwaardeproblemen, voor het specifieke geval dat N = 2 en M = 0, i.e., het linkerlid van de vergelijking bevat afgeleiden van de output y (t ) tot en met de tweede afgeleide terwijl het rechterlid van de vergelijking enkel de input x(t ) bevat zonder afgeleiden:
a2 y ''(t ) + a1 y '(t ) + a0 y (t ) = b0x(t )
met beginvoorwaarden die worden opgelegd voor y (0) en y '(0). Merk op dat het gebruikelijk is om de vergelijking te herschalen zodat a2 = 1 en om de coëfficiënt b0 te "absorberen" in het inputsignaal x(t ), i.e.,
y ''(t ) + a1 y '(t ) + a0 y (t ) = x(t )
    We vatten nu de oplossingsstrategie samen. Voor een meer gedetailleerde behandeling verwijzen we naar het vak Wiskundige Modellen.

2.1.2 Particuliere oplossing
De tweede stap voor het vinden van de algemene oplossing van een LDV met constante coëfficieënten, bestaat erin een particuliere oplossing te zoeken. Dit is een oplossing yp (t ) die aan de LDV voldoet, maar die niet de meest algemene vorm voorstelt die eender welke oplossing y (t ) van de LDV kan aannemen.
    Er bestaan verschillende methoden om een particuliere oplossing te vinden. Hier zullen we enkel de methode van de onbepaalde coëfficieënten toelichten. Deze methode is toepasbaar op LDVs met ingangssignalen x(t ) waarvan de afgelei- den binnen dezelfde "familie" van functies blijven, e.g.,
• veeltermfuncties (waarvan de afgeleiden opnieuw veeltermfuncties zijn)

• exponentiële functies (waarvan de afgeleiden opnieuw exponentiële func- ties zijn)
• sinus- en cosinusfuncties (waarvan de afgeleiden opnieuw sinus- en cos- inusfuncties zijn)
De methode van de onbepaalde coëfficieënten is ook toepasbaar wanneer het rechterlid uit een lineaire combinatie of een product van bovenstaande functies bestaat.
